PASO 1: desde terminal navegar hasta la carpeta raiz donde se encuentra archivo .Java usando comando:
 - cd 'ruta de carpeta'
  - EJEMPLO: cd '/Users/test/IdeaProjects/Tasca_S1.05/src/main/java

PASO 2: compilar las clases con comando:
 - javac 'ruta desde archivo raiz'/clase.java
  - EJEMPLO: javac org/example/n1exercici1_4/Main.java org/example/n1exercici1_4/Main.java

PASO 3: ejecutar Main:
 - java rutadesde archivo raiz/clase (sin extencion)
  - EJEMPLO: java org/example/n1exercici1_4/Main
